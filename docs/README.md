# ResuMake

ResuMake is a command-line resume content manager for students and early-career job applicants.
It helps users store, edit, and organize projects, experiences, and CCAs in a structured format,
so resume-ready content can be prepared quickly without using form-heavy editors.

## Key features

- Add and manage timed records (`project`, `experience`, `cca`) with role, tech stack, and date range.
- After adding a record, optionally capture bullets immediately (`y`/`n`; stop with `esc`).
- Add, edit, move, find, and delete bullet points under each record.
- Search records and bullets using keywords (`find`, `findbullet`).
- Edit user profile fields (`name`, `number`, `email`) used in generated output.
- Reject duplicate records and duplicate bullets.
- Generate a grouped resume view with user profile details and a derived skills section.
- Persistent storage with automatic save/load from `records.txt`, with write-skips when unchanged.
